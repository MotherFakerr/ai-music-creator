# 里程碑2：音乐跟奏

> 状态：进行中

## 目标

用户可以上传自己的音乐，在音乐播放的同时通过键盘进行演奏。

## 核心功能

| 功能 | 描述 |
|-----|------|
| 音频上传 | 支持上传音频文件（mp3/wav） |
| 音频播放 | 播放器组件 + 播放/暂停 |
| 键盘演奏 | 在播放过程中实时键盘弹奏 |

## 拓展功能

| 功能 | 描述 |
|-----|------|
| 循环播放 | 设置 A/B 点循环练习 |
| 速度控制 | 0.5x ~ 2x 变速 |
| 音量控制 | 分别调节伴奏与演奏音量 |
| 录音导出 | 录制混合结果并下载 |

## 不做

- ❌ 自动伴奏生成
- ❌ 音频编辑
- ❌ 节拍检测（v1）

## 成功标准

- 能上传并播放音频
- 播放时键盘可正常演奏
- 音频与键盘声音混合输出
