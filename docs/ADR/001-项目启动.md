# ADR-001: 项目启动与技术栈初选

## 状态

✅ 已确认

## 决策

选择 **TypeScript + Monorepo** 作为项目基础技术栈。

## 背景

主人是5年前端开发者，熟悉 Web 技术栈。项目定位是长期演进工具，需要良好的类型安全和代码组织。

## 决策理由

| 技术 | 选择理由 |
|-----|---------|
| TypeScript | 类型安全对长期演进至关重要，减少隐性错误 |
| Monorepo | 音乐软件天然需要分离多个模块（演奏/跟奏/AI），Monorepo 便于共享代码和类型 |

## 未决策项

以下内容待后续 ADR 决定：

- 前端框架（React / Vue / Svelte）
- 音频引擎（Web Audio API / Tone.js）
- MIDI 处理库
- 状态管理方案
- AI 接入方式

## 约束条件

- 必须是前端技术（Web）
- 需要支持长期演进
- vibe coding 优先，快速原型
